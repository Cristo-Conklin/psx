1.11. PSX_Html_Lexer

Robust html5 parser to parse non well-formed markup. It splits the html in to an element / text token stream wich is pushed into an dom builder who builds the tree structure depending on the incomming tokens. If a closing tag is missing the dom build automatically closes the tag. Here an example howto get all links from an html markup. $root = PSX_Html_Lexer::parse($html); $elements = $root->getElementsByTagName('a'); foreach($elements as $el) { $href = $el->getAttribute('href'); if($href !== false) { echo $href . "\n"; } }
