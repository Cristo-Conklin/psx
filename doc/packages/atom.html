1.1. PSX_Atom

This class is for converting an raw ATOM XML feed into an readable object. Where you can access all values without parsing the XML. You can import an existing feed with the PSX_Data_Reader_Dom. Here an example howto read an external atom feed. $atom = PSX_Atom::request('http://test.phpsx.org/index.php/atom'); echo $atom->title: echo $atom->updated; foreach($atom as $entry) { echo $entry->title . "\n"; } If your module extends the PSX_Module_ApiAbstract class you can parse an atom feed wich is send to your via an HTTP request with the getRequest() method $atom = new PSX_Atom(); $atom->import($this->getRequest(PSX_Data_ReaderInterface::DOM)); echo $atom->title: echo $atom->updated;
